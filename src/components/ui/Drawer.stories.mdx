# Drawer Component

A comprehensive off-canvas drawer system built with Radix UI Dialog primitives and class-variance-authority for type-safe variants.

## Features

- **Multiple Positions**: Left, right, top, bottom drawer positions
- **Multiple Sizes**: Six size variants from xs to full-width/height
- **Specialized Components**: NavigationDrawer, RecipeDrawer, MobileMenuDrawer
- **Recipe-Specific Features**: Ingredient checklists, cooking instructions, shopping lists
- **Full Accessibility**: ARIA attributes, keyboard navigation, focus management, screen reader support
- **Animation Support**: Smooth slide-in/out transitions with proper state management
- **Overlay Controls**: Configurable background overlay with click-to-close
- **Type Safety**: Full TypeScript support with comprehensive type definitions
- **Customizable**: Easy to extend with custom classes and content

## Import

```tsx
import {
  Drawer,
  DrawerTrigger,
  DrawerOverlay,
  DrawerContent,
  DrawerHeader,
  DrawerBody,
  DrawerFooter,
  DrawerClose,
  DrawerTitle,
  DrawerDescription,
  NavigationDrawer,
  RecipeDrawer,
  MobileMenuDrawer,
} from '@/components/ui/drawer';
```

## Basic Usage

```tsx
// Basic drawer
<Drawer>
  <DrawerTrigger asChild>
    <Button>Open Drawer</Button>
  </DrawerTrigger>
  <DrawerOverlay />
  <DrawerContent position="right" size="md">
    <DrawerHeader>
      <DrawerTitle>Drawer Title</DrawerTitle>
      <DrawerDescription>Drawer description</DrawerDescription>
    </DrawerHeader>
    <DrawerBody>
      <p>Drawer content goes here</p>
    </DrawerBody>
    <DrawerFooter>
      <DrawerClose asChild>
        <Button variant="outline">Close</Button>
      </DrawerClose>
    </DrawerFooter>
  </DrawerContent>
</Drawer>
```

    icon: 'home',
    href: '/',

  },

    href: '/recipes',

  },
  {

## Controlled vs Uncontrolled

### Controlled
Manage the open state externally.

```tsx
const [open, setOpen] = useState(false);

<Drawer open={open} onOpenChange={setOpen}>
  <DrawerTrigger asChild>
    <Button>Open Controlled</Button>
  </DrawerTrigger>
  <DrawerContent>
    <DrawerHeader>
      <DrawerTitle>Controlled Drawer</DrawerTitle>
    </DrawerHeader>
  </DrawerContent>
</Drawer>
```

### Uncontrolled
Let the component manage its own state.

```tsx
<Drawer defaultOpen={false}>
  <DrawerTrigger asChild>
    <Button>Open Uncontrolled</Button>
  </DrawerTrigger>
  <DrawerContent>
    <DrawerHeader>
      <DrawerTitle>Uncontrolled Drawer</DrawerTitle>
    </DrawerHeader>
  </DrawerContent>
</Drawer>
```

## Accessibility

- **Keyboard Navigation**: Full keyboard support with proper focus management
- **Screen Reader Support**: ARIA labels, descriptions, and announcements
- **Focus Trapping**: Focus is trapped within the drawer when open
- **Focus Restoration**: Focus returns to trigger element when closed
- **Escape Key**: Closes drawer when escape is pressed (configurable)
- **Overlay Click**: Closes drawer when overlay is clicked (configurable)

## Recipe App Patterns

### Navigation Pattern
Use NavigationDrawer for main app navigation with recipe categories and user actions.

### Ingredient Management
Use RecipeDrawer with ingredient type for shopping list management and cooking preparation.

### Cooking Mode
Use RecipeDrawer with instructions type for step-by-step cooking guidance with progress tracking.

### Mobile Experience
Use MobileMenuDrawer for responsive navigation with search functionality.

## Props Reference

### Drawer Props
- `open?: boolean` - Controls drawer open state
- `onOpenChange?: (open: boolean) => void` - Callback when open state changes
- `defaultOpen?: boolean` - Default open state for uncontrolled usage
- `showOverlay?: boolean` - Whether to show background overlay
- `closeOnOverlayClick?: boolean` - Close when overlay is clicked
- `closeOnEscape?: boolean` - Close when escape key is pressed
- `preventScroll?: boolean` - Prevent body scrolling when open

### DrawerContent Props
- `position?: 'left' | 'right' | 'top' | 'bottom'` - Drawer position
- `size?: 'xs' | 'sm' | 'md' | 'lg' | 'xl' | 'full'` - Drawer size
- `variant?: 'default' | 'elevated' | 'minimal' | 'overlay'` - Style variant
- `showClose?: boolean` - Show close button in header

### NavigationDrawer Props
- `items: NavigationItem[]` - Array of navigation items
- `activeItemId?: string` - Currently active item ID
- `showProfile?: boolean` - Show user profile section
- `userProfile?: object` - User profile data
- `onItemClick?: (item: NavigationItem) => void` - Item click handler

### RecipeDrawer Props
- `recipeTitle?: string` - Recipe title to display
- `ingredients?: DrawerIngredient[]` - Array of recipe ingredients
- `instructions?: DrawerInstruction[]` - Array of cooking instructions
- `shoppingList?: ShoppingListItem[]` - Array of shopping list items
- `type?: 'ingredients' | 'instructions' | 'shopping' | 'notes'` - Content type
- `showIngredientChecklist?: boolean` - Enable ingredient checking
- `showInstructionProgress?: boolean` - Show instruction progress
- `onIngredientCheck?: (id: string, checked: boolean) => void` - Ingredient check handler
- `onInstructionComplete?: (id: string, completed: boolean) => void` - Instruction complete handler
- `onShoppingItemCheck?: (id: string, checked: boolean) => void` - Shopping item check handler
