## Usage Guidelines

### When to use
- For grouping related content and actions
- To create visual hierarchy and organization
- For displaying recipes, user profiles, or content previews
- In dashboards and content galleries
- For interactive content that needs hover/focus states

### Best practices
- Use appropriate variants for visual hierarchy
- Combine with other components like Button, Badge, Avatar
- Keep content focused and related
- Use interactive variant for clickable cards
- Maintain consistent spacing with the size prop
- Use semantic heading levels in CardTitle
- Provide meaningful descriptions in CardDescription

### Accessibility
- Cards are semantically neutral containers
- Use proper heading hierarchy within cards
- Interactive cards should have proper focus indicators
- Include appropriate ARIA labels for complex interactions
- Ensure sufficient color contrast in all variants
