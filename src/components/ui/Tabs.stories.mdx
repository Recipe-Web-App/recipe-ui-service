## Best Practices

### Recipe App Usage

1. **Content Organization**: Use tabs to organize complex recipe information
2. **User Experience**: Keep tab labels short and descriptive
3. **Mobile Consideration**: Use horizontal scrolling for many tabs on mobile
4. **Consistent Variants**: Stick to one variant style within the same interface
5. **Loading States**: Show loading indicators in tab content when needed

### Implementation Guidelines

```tsx
// Good: Clear, descriptive tab labels
<TabsTrigger value="ingredients">Ingredients</TabsTrigger>
<TabsTrigger value="instructions">Instructions</TabsTrigger>

// Good: Consistent variant usage
<TabsList variant="line">
  <TabsTrigger variant="line" value="tab1">Tab 1</TabsTrigger>
  <TabsTrigger variant="line" value="tab2">Tab 2</TabsTrigger>
</TabsList>

// Good: Proper content organization
<TabsContent value="ingredients">
  <IngredientsList />
</TabsContent>
```
