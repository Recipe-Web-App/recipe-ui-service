## Accessibility

The CommandPalette component includes comprehensive accessibility features:

- **Keyboard Navigation**: Full keyboard support with arrow keys, Enter, Escape, Home, End
- **ARIA Attributes**: Proper combobox, listbox, and option roles with ARIA labels
- **Focus Management**: Maintains focus within the modal and returns focus appropriately
- **Screen Reader Support**: Descriptive labels and announcements for all interactions
- **Portal Rendering**: Renders above all content with proper modal behavior

## Keyboard Shortcuts

The component supports these keyboard interactions:

- **Arrow Up/Down**: Navigate through commands
- **Home/End**: Jump to first/last command
- **Enter**: Execute the highlighted command
- **Escape**: Close the command palette
- **Type to search**: Filter commands in real-time

## Performance

The component is optimized for performance:

- **Virtual Scrolling**: Efficient rendering of large command lists
- **Fuzzy Search**: Fast search algorithm with scoring and ranking
- **Memoized Filtering**: Optimized re-renders with React.memo patterns
- **Portal Rendering**: Minimal DOM manipulation with React portals
- **Bundle Size**: Lightweight implementation with minimal dependencies

## Integration Examples

### Global Keyboard Shortcut

\`\`\`tsx
useEffect(() => {
const handleKeyDown = (event: KeyboardEvent) => {
if (event.key === 'k' && (event.metaKey || event.ctrlKey)) {
event.preventDefault();
setCommandPaletteOpen(true);
}
};

document.addEventListener('keydown', handleKeyDown);
return () => document.removeEventListener('keydown', handleKeyDown);
}, []);
\`\`\`

### Recent Commands Tracking

\`\`\`tsx
const [recentCommands, setRecentCommands] = useState<Command[]>([]);

const handleCommandSelect = (command: Command) => {
// Execute command
command.action();

// Track as recent (remove duplicates, limit to 5)
setRecentCommands(prev => {
const filtered = prev.filter(c => c.id !== command.id);
return [command, ...filtered].slice(0, 5);
});
};
