## Usage Guidelines

### When to Use

- **Textarea**: General multi-line text input needs
- **RecipeTextarea**: Recipe-specific content with contextual styling
- **AutoTextarea**: When you want automatic height adjustment

### Recipe Type Guidelines

- **description**: Recipe overview, inspiration, or backstory (blue theme)
- **instructions**: Step-by-step cooking directions (green theme)
- **notes**: Chef's notes, tips, or substitutions (yellow theme)
- **tips**: Pro tips and cooking tricks (purple theme)
- **review**: User reviews and experiences (orange theme)

### Best Practices

1. Always provide clear labels and helpful placeholder text
2. Use appropriate validation with helpful error messages
3. Consider character/word limits for better user experience
4. Use auto-resize for dynamic content that varies in length
5. Provide context-specific helper text
6. Consider the recipe type for appropriate styling and validation
