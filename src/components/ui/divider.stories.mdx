# Divider Component

A flexible, accessible divider component with support for horizontal/vertical orientations, various styles, text/icon decorations, and recipe-specific variants. Built with TypeScript and full accessibility compliance.

## Features

- **Multiple Orientations**: Horizontal and vertical dividers
- **Style Options**: Solid, dashed, dotted, and double line styles
- **Weight Variants**: Thin, normal, and thick dividers
- **Length Options**: Short, normal, long, and full width/height
- **Color Variants**: Multiple color options including primary, secondary, and semantic colors
- **Text Integration**: Dividers with centered, start, or end-positioned text
- **Icon Support**: Dividers with icons and custom styling
- **Recipe-Specific**: Specialized dividers for recipe sections and content groups
- **Section Dividers**: Enhanced dividers with titles, subtitles, and collapsible functionality
- **Full Accessibility**: Proper ARIA attributes and semantic markup

## Orientations

### Horizontal (Default)
Standard horizontal divider for separating content vertically.

```tsx

  <p>Content above the divider</p>
```

```tsx
<div className="flex h-32 items-center gap-4">
```

## Advanced Usage

### Interactive Divider Controls
Create dynamic dividers with user controls.

```tsx
const [selectedStyle, setSelectedStyle] = useState('solid');
const [selectedWeight, setSelectedWeight] = useState('thin');
const [selectedColor, setSelectedColor] = useState('default');

return (
  <div className="space-y-8">
    <div className="grid grid-cols-3 gap-4">
      <div>
        <label className="mb-2 block text-sm font-medium">Style</label>
        <select
          value={selectedStyle}
          onChange={(e) => setSelectedStyle(e.target.value)}
          className="w-full rounded border px-3 py-2"
        >
          <option value="solid">Solid</option>
          <option value="dashed">Dashed</option>
          <option value="dotted">Dotted</option>
          <option value="double">Double</option>
        </select>
      </div>

      <div>
        <label className="mb-2 block text-sm font-medium">Weight</label>
        <select
          value={selectedWeight}
          onChange={(e) => setSelectedWeight(e.target.value)}
          className="w-full rounded border px-3 py-2"
        >
          <option value="thin">Thin</option>
          <option value="normal">Normal</option>
          <option value="thick">Thick</option>
        </select>
      </div>

      <div>
        <label className="mb-2 block text-sm font-medium">Color</label>
        <select
          value={selectedColor}
          onChange={(e) => setSelectedColor(e.target.value)}
          className="w-full rounded border px-3 py-2"
        >
          <option value="default">Default</option>
          <option value="primary">Primary</option>
          <option value="secondary">Secondary</option>
        </select>
      </div>
    </div>

    <div className="mt-8">
      <p className="mb-4">Live Preview:</p>
      <Divider
        style={selectedStyle}
        weight={selectedWeight}
        color={selectedColor}
      />
    </div>
  </div>
);
```

## Accessibility

The Divider component follows WCAG accessibility guidelines:

### Semantic Markup
- Uses proper `role="separator"` for dividers
- Decorative dividers marked with `aria-hidden="true"`
- Semantic dividers include appropriate ARIA labels

### Screen Reader Support
- Text and icon content properly announced
- Section dividers include heading semantics
- Collapsible sections communicate state changes

### Focus Management
- Interactive section dividers are keyboard accessible
- Proper focus indicators for collapsible elements
- Logical tab order maintained

### Color and Contrast
- Sufficient color contrast for all variants
- Visual information not conveyed by color alone
- High contrast mode support

## Design Guidelines

### When to Use Each Style
- **Solid**: Standard separation, formal content, clean interfaces
- **Dashed**: Casual separation, draft content, temporary sections
- **Dotted**: Subtle separation, decorative elements, light divisions
- **Double**: Formal separation, important sections, emphasis

### Weight Guidelines
- **Thin**: Subtle separation, dense layouts, minimal visual disruption
- **Normal**: Standard separation, balanced visual hierarchy
- **Thick**: Strong separation, section boundaries, emphasis

### Color Guidelines
- **Default**: Neutral separation, follows theme colors
- **Muted**: Subtle separation, background content
- **Primary**: Important sections, brand-related content
- **Secondary**: Alternative sections, secondary content
- **Accent**: Highlighted sections, special content
- **Destructive**: Warning sections, error content

### Length Guidelines
- **Short**: Decorative separation, inline content, subtle breaks
- **Normal**: Standard content separation, balanced layouts
- **Long**: Prominent separation, section boundaries
- **Full**: Complete separation, page sections, strong boundaries

## Props Reference

### Divider Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | Divider orientation |
| `style` | `'solid' \| 'dashed' \| 'dotted' \| 'double'` | `'solid'` | Line style |
| `weight` | `'thin' \| 'normal' \| 'thick'` | `'thin'` | Line thickness |
| `length` | `'short' \| 'normal' \| 'long' \| 'full'` | `'full'` | Divider length |
| `spacing` | `'tight' \| 'normal' \| 'loose'` | `'normal'` | Spacing around divider |
| `color` | `'default' \| 'muted' \| 'primary' \| 'secondary' \| 'accent' \| 'destructive'` | `'default'` | Color variant |
| `decorative` | `boolean` | `true` | Whether divider is decorative (affects ARIA) |
| `className` | `string` | - | Additional CSS classes |

### DividerWithText Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `text` | `string` | - | Text content |
| `textPosition` | `'start' \| 'center' \| 'end'` | `'center'` | Text position |
| `textProps` | `TextProps` | - | Text styling props |
| All base Divider props | - | - | Inherited from Divider |

### DividerWithIcon Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `icon` | `ReactNode` | - | Icon element |
| `textPosition` | `'start' \| 'center' \| 'end'` | `'center'` | Icon position |
| `iconProps` | `IconProps` | - | Icon styling props |
| All base Divider props | - | - | Inherited from Divider |

### RecipeDivider Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `context` | `'ingredient-group' \| 'instruction-step' \| 'nutrition-group' \| 'recipe-section'` | - | Recipe context |
| `label` | `string` | - | Optional label text |
| `showLabel` | `boolean` | `false` | Whether to show label |
| All base Divider props | - | - | Inherited from Divider |

### SectionDivider Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `section` | `'ingredients' \| 'instructions' \| 'nutrition' \| 'metadata'` | - | Section type |
| `title` | `string` | - | Section title |
| `subtitle` | `string` | - | Optional subtitle |
| `icon` | `ReactNode` | - | Optional icon |
| `collapsible` | `boolean` | `false` | Whether section is collapsible |
| `onToggleCollapse` | `(collapsed: boolean) => void` | - | Collapse toggle handler |
| All base Divider props | - | - | Inherited from Divider |

Plus all standard HTML div attributes.
